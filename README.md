# Simple proxy convert subtitle from SRT to VTT

## Overview
- Api: https://convert-subtitle-srt-to-vtt.vercel.app
- Author: Nguyễn Hoàng Lâm ([@lamhoang1256](https://github.com/lamhoang1256))
